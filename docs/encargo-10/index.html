Encargo 10

Para este décimo encargo se desarrolló los modelos 3D según los tutoriales expuestos en la web del ramo “Taller Integrado”.

Tutorial

Se hizo una línea en un sketch en “surface” y luego, se ha extruido (diapositiva 1). De la misma forma, se realizan dos líneas y se extruyen (diapositiva 2). Luego, se cortan con “trim” y se hace lo mismo con un sketch que se realiza por sobre la figura hecha (diapositiva 3). Estas figuras (cuando ya se han cortado), se les hace un “offset surface” y se escalan (diapositiva 4). Finalmente, se realiza un “loft” entre las superficies sueltas, proporcionándoles también “surface stitch”, el cual funciona como un “join” entre las superficies. Para terminar la figura, se realizan diferentes “fillet” (diapositiva 5).
